# Binary Addition

### Description

Implement a function that adds two numbers together and returns their sum in binary. The conversion can be done before, or after the addition.

The binary number returned should be a string.

### Examples

```javascript
addBinary(1, 1); // '10'
addBinary(5, 9); // '1110'
```
